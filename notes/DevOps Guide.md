# DevOps Guide

## CI/CD Pipeline

Our continuous integration and deployment workflow for [[Project Alpha]].

### Pipeline Stages

1. **Lint & Test** - Run on every push
2. **Build** - Docker image creation
3. **Staging Deploy** - Auto-deploy to staging
4. **Production Deploy** - Manual approval required
